howto: GitHub hello-world repository from the command line

  1. To begin, create a new directory:
mkdir hello-world
cd hello-world
  1. Create a new repository:
git init
  1. Create a new README.md file to add project info:
vim README.md

# hello-world

Hi humanoids.
This is one of the first steps into conquering the digital world.

See you there.
  1. Add file to git
git add README.md
  1. Commit changes to repo
git commit -m "First commit"
  1. Add GitHub remote repo
git remote add origin https://github.com/cesmus/hello-world.git
  1. Push changes to GitHub:
git push
